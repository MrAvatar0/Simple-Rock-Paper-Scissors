

# **Rock–Paper–Scissors (HTML / JavaScript)**

A simple **Rock–Paper–Scissors** game built with **HTML, CSS, and JavaScript**.
In this project, you play against the **computer (CPU)**. The CPU randomly selects Rock, Paper, or Scissors, and the game instantly shows the result.

## **Features**

* Play against a fully random CPU opponent
* Clean and simple UI
* Real-time result display
* Lightweight — no frameworks required
* Works in any modern browser

## **How to Run**

1. Download or clone the repository.
2. Open the `index.html` file in your browser (Chrome, Firefox, etc.).
3. Choose your move and watch the CPU respond.
